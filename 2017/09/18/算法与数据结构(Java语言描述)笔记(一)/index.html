模式匹配

如果子串T在主串中存在，则返回存在的位置，如果不存在，则返回-1。

简单模式匹配

从主串的第pos位置字符开始和模式子串字符比较，如果相等，则继续逐个比较后续字符；否则从主串的下一个字符起再重新和模式子串的字符比较。直到找到匹配字符串或者是主串结尾。
